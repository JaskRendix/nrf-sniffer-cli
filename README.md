# nrf-sniffer-cli

This package provides Python APIs and a simple command‑line tool for working with the **Nordic nRF Sniffer for Bluetooth LE**.

> **Originally based on:**  
> The open‑source project **embedded-community/nrf-sniffer-cli**  
> GitHub: [https://github.com/embedded-community/nrf-sniffer-cli](https://github.com/embedded-community/nrf-sniffer-cli)  
>  
> The `SnifferAPI/` directory contains code copied from Nordic Semiconductor’s official nRF Sniffer package.

## Basic Usage

Install directly from GitHub:

```
pip install git+https://github.com/embedded-community/nrf-sniffer-cli.git
```

Scan for advertising BLE devices:

```
nrf-sniffer-cli scan
```

Start sniffing a specific device by address:

```
nrf-sniffer-cli sniff --address 001122334455
```

Or by name:

```
nrf-sniffer-cli sniff --name example-name
```

By default, packets are written to `capture.pcap`. You can override this:

```
nrf-sniffer-cli --capture-file example-capture.pcap sniff --name example-name
```

Stop sniffing with **Ctrl+C**.

## Optional Analysis Tools (`tools/`)

These scripts consume NDJSON output generated by:

```
nrf-sniffer-cli sniff --record-json packets.json
```

They are **not part of the CLI** and **do not affect the package**.  
They are intended for debugging, RF analysis, and BLE sensor evaluation.

### Included Tools

| Tool | Description |
|------|-------------|
| **tail_rssi.py** | Live RSSI bar graph in the terminal. |
| **channel_heatmap.py** | Real‑time distribution of packets across BLE channels 37/38/39. |
| **monitor_battery.py** | Extracts Battery Service (UUID 0x2A19) notifications. |
| **monitor_gaps.py** | Detects connection dropouts by measuring timestamp gaps. |
| **monitor_stability.py** | Measures packets‑per‑second to detect RF interference. |
| **parse_sensor_data.py** | Extracts temperature/humidity from Xiaomi/Govee/RuuviTag advertisements. |
| **watch_target.py** | Alerts when a specific device name appears in advertising packets. |
| **watchdog_device_offline.py** | Alerts when a device disappears for >10 seconds. |
| **tail_with_rotation.py** | Rotation‑aware tail script for long‑running sniffers. |
| **web_dashboard/** | Flask + SocketIO real‑time dashboard with live RSSI graph. |

## Web Dashboard

Located in:

```
tools/web_dashboard/
```

Features:

- Live RSSI gauge  
- 60‑second rolling signal graph (Chart.js)  
- Device name display  
- Mobile‑friendly UI  

Run it:

```
python tools/web_dashboard/app.py
```

Then open:

```
http://<your-ip>:5000
```
